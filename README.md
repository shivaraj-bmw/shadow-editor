# 🚀 SmartLayer AI Editor

SmartLayer AI Editor is a Flutter-based advanced image editing application featuring a professional layer system, AI-powered editing tools, and real-time image manipulation.

## ✨ Features

* Multi-Layer Editing
* Drag, Resize, Rotate & Scale
* AI Background Removal
* AI Image Generator
* AI Sticker Generator
* Real Crop Tool
* Text Effects (Gradient, Glow, Shadow, Stroke)
* Image Filters
* Undo / Redo System
* Layer Lock & Layer Management
* Shape Layers
* Gesture-Based Editing

## 🛠 Technologies

* Flutter
* Dart
* ONNX Runtime
* AI Models
* Image Processing

## 🚀 Upcoming Features

* HD Upscale
* Magic Eraser
* Manual Background Removal
* Advanced AI Tools
